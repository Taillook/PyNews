PyNews
====

Googleニュースの記事を引っ張ってくる.

## Description
Googleニュースから特定のトピックから記事を抽出する．

## Requirement

- Python 3.5
- requests
- argparse

## Usage
`python get_news.py --topic h --amount 20`

### Options

- --topic h([トピック一覧](http://so-zou.jp/web-app/tech/web-api/google/search/news/parameter.htm#topic))
- --amount 取得数(最大20)
